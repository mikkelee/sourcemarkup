sourcemarkup by carthag

Cocoa project to ease transcription of images of genealogical records; as well as provide a documented XML format for exchange of said transcriptions.

Currently very bare bones frontend-wise, most work is done in backend, front only exists to make it possible for me to test the model.

Please see source1.xml and source2.xml for examples for the format (work in progress).
